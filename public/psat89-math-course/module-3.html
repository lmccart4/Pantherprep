Topic 3A

Exponent Rules

These six rules show up constantly. Memorize them and you'll handle about a third of all Advanced Math questions automatically.

3A
The Six Exponent Rules
Product, quotient, power, zero, negative, and distribution
▸
Product Rule: x^a · x^b = x^(a+b)
Quotient Rule: x^a ÷ x^b = x^(a−b)
Power Rule: (x^a)^b = x^(a×b)
Zero Rule: x^0 = 1 (anything to the 0 is 1)
Negative Rule: x^(−n) = 1/x^n
Distribution: (xy)^n = x^n · y^n

Common Mistakes

  • Adding exponents when multiplying: x² · x³ = x⁵ ✓ (not x⁶)
  • Multiplying exponents for power rule: (x²)³ = x⁶ ✓ (not x⁵)
  • Can't add different bases: x² + y² stays as x² + y² (can't combine!)
  • Negative exponent ≠ negative number: 3⁻² = 1/9, not −9
Topic 3B

Quadratic Equations — The Three Forms

Every quadratic on the PSAT 8/9 can be expressed in three forms. Each form reveals different information.

3B
Standard, Vertex & Factored Forms
What each form tells you and when to use it
▸
Standard Form: f(x) = ax² + bx + c
Reveals: y-intercept (c), direction (a>0 opens up, a<0 opens down)
Vertex Form: f(x) = a(x − h)² + k
Reveals: vertex at (h, k), axis of symmetry x = h
Factored Form: f(x) = a(x − r₁)(x − r₂)
Reveals: x-intercepts (zeros) at r₁ and r₂

Solving Quadratics

  • Factoring: If x² − 5x + 6 = 0, find two numbers that multiply to 6 and add to −5 → (x−2)(x−3) = 0
  • Square root method: If (x − 3)² = 16, then x − 3 = ±4, so x = 7 or x = −1
  • Quadratic formula: x = (−b ± √(b²−4ac)) / 2a — works on everything, use as backup

The Discriminant

b² − 4ac > 0: Two real solutions
b² − 4ac = 0: Exactly one real solution
b² − 4ac < 0: No real solutions

Vertex Shortcut

For ax² + bx + c: x-coordinate of vertex = −b/(2a). Plug back in for y.

Desmos Power Move: Type any quadratic and Desmos instantly shows the parabola. Click the vertex and intercepts for exact values.
Topic 3C

Polynomials & Factoring

3C
Polynomial Operations & Factoring Techniques
Adding, subtracting, multiplying polynomials; GCF, trinomial, difference of squares
▸

Operations

  • Adding/subtracting: Combine like terms only. 3x² + 2x² = 5x², but x² + x stays as is.
  • Multiplying: FOIL for binomials, or distribute each term. (2x+3)(x−4) = 2x²−8x+3x−12 = 2x²−5x−12
  • No long division on PSAT 8/9 — just the basics.

Three Factoring Techniques

GCF First: 6x² + 12x = 6x(x + 2)
Trinomial: x² + 5x + 6 = (x + 2)(x + 3) — find two numbers that multiply to c and add to b
Difference of Squares: x² − 9 = (x + 3)(x − 3)
Always factor GCF first. It makes everything else easier. If there's no GCF, try trinomial factoring. If it's a² − b², use difference of squares.
Topic 3D

Functions, Transformations & Exponential Models

3D
Function Notation, Transformations & Growth/Decay
Evaluating, interpreting, and recognizing function behavior
▸

Function Notation

  • f(x) = 2x + 3 means: plug in a value for x, get a value back. f(5) = 2(5) + 3 = 13.
  • Domain: all valid inputs (x-values). Range: all possible outputs (y-values).
  • If f(a) = 0, then a is a zero (or root) of the function.

Basic Transformations

f(x) + k: shifts UP k units
f(x) − k: shifts DOWN k units
f(x − h): shifts RIGHT h units
f(x + h): shifts LEFT h units
−f(x): reflects over x-axis (flips upside down)
Horizontal shifts feel backwards: f(x − 3) shifts right, not left. Think of it as: what value of x makes the inside equal to 0? x = 3 → shifts to 3.

Exponential Growth & Decay

Growth: y = a(1 + r)^t — multiplier > 1
Decay: y = a(1 − r)^t — multiplier between 0 and 1
a = initial amount, r = rate (as decimal), t = time
  • Population doubles every 5 years: y = P₀ · 2^(t/5)
  • Value decreases 20% per year: y = V₀ · (0.80)^t
  • Key insight: If the base (multiplier) is > 1, it's growth. If 0 < base < 1, it's decay.
Math Trap Taxonomy

Four Traps in Advanced Math

These traps from Module 2 show up again here — plus one new one specific to quadratics. Spot them before they spot you.

1
🎯 The Wrong Target
Solved for x when they asked for x² + 1 or f(3). Always underline what the question asks for.
Asked: "What is 2x + 1?" You solved x = 3 and picked 3. Answer: 7
2
👻 The Distribution Ghost
Forgot to distribute to ALL terms during FOIL or expansion. The ghost term gets left behind.
(x+3)(x−2) = x²−2x+3x−6 = x²+x−6 (not x²+x+6)
3
🔀 The One-Case Trap
Only solved one branch of a square root or absolute value. Remember: x² = 9 means x = ±3.
(x−1)² = 4 → x−1 = ±2 → x = 3 AND x = −1 (two answers)
4
📐 The Form Mismatch
Used the wrong quadratic form for what the question asks. Vertex question? Use vertex form. Zeros? Use factored form.
Asked for vertex but used standard form and forgot −b/(2a). Use vertex form: (h,k) is right there.
