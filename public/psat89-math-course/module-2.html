Core Skills

  • Solving one-variable equations: Isolate the variable using inverse operations. Whatever you do to one side, do to the other.
  • Number of solutions: One solution (normal), no solution (contradiction like 3 = 5), infinitely many (identity like 2x = 2x)
  • Word problem translation: "is" ‚Üí =, "more than" ‚Üí +, "less than" ‚Üí ‚àí, "per" ‚Üí √ó (rate), "of" ‚Üí √ó
  • Absolute value: |ax + b| = c produces TWO cases: ax + b = c AND ax + b = ‚àíc
  • Interpreting in context: In C = 50 + 0.15m, the 50 is the flat fee and 0.15 is the per-mile rate
Solve: Distribute ‚Üí combine like terms ‚Üí isolate variable ‚Üí check
No solution: Variables cancel, leaves false statement (e.g., 0 = 5)
Infinite solutions: Variables cancel, leaves true statement (e.g., 0 = 0)

Systems of Linear Equations

Three Methods

  • Substitution: Solve one equation for a variable, plug into the other.
  • Elimination: Add/subtract equations to cancel a variable.
  • Graphing (Desmos): Graph both, click the intersection. Often fastest.
One solution: Different slopes ‚Üí lines cross
No solution: Same slope, different intercepts ‚Üí parallel
Infinite solutions: Equations are multiples ‚Üí same line

Summary

Key Takeaways

  1. Algebra is ~35% of the PSAT 8/9 ‚Äî the single largest domain.
  2. Know all three forms: slope-intercept, point-slope, standard.
  3. Parallel = same slope. Perpendicular = negative reciprocal slopes.
  4. Systems: substitution, elimination, or Desmos for speed.
  5. Variables cancel ‚Üí true = infinite solutions, false = no solution.
  6. Inequalities: FLIP the sign when multiplying/dividing by a negative.
  7. 50%+ of word problem errors happen at translation. Practice converting English to equations.
  8. Always re-read what the question asks for before choosing your answer.
