The Three Forms

Slope-Intercept: y = mx + b
m = slope (rate of change), b = y-intercept (starting value)
Point-Slope: y ‚àí y‚ÇÅ = m(x ‚àí x‚ÇÅ)
Use when given a point and slope
Standard Form: Ax + By = C
Useful for finding intercepts (set x=0 or y=0) and for systems

Slope

  • Formula: m = (y‚ÇÇ ‚àí y‚ÇÅ) / (x‚ÇÇ ‚àí x‚ÇÅ) ‚Äî rise over run
  • Positive slope: line goes up left to right ‚Üó
  • Negative slope: line goes down left to right ‚Üò
  • Zero slope: horizontal line (y = constant)
  • Undefined slope: vertical line (x = constant)

Special Relationships

  • Parallel lines: Same slope, different y-intercepts (m‚ÇÅ = m‚ÇÇ)
  • Perpendicular lines: Slopes are negative reciprocals (m‚ÇÅ √ó m‚ÇÇ = ‚àí1)

Systems of Linear Equations
Substitution, elimination, number of solutions, word problems
‚ñ∏

Three Methods

  • Substitution: Solve one equation for a variable, plug into the other. Best when one variable is already isolated (e.g., y = 3x + 1).
  • Elimination: Add or subtract equations to cancel a variable. Best when coefficients are easy to align.
  • Graphing (Desmos): Graph both equations, click the intersection point. Often the fastest method on the PSAT.

Number of Solutions

One solution: Lines intersect at one point (different slopes)
No solution: Lines are parallel (same slope, different intercepts)
Infinite solutions: Lines are the same (equations are multiples of each other)

Context Problems

  • Mixture problems: Two equations ‚Äî one for quantity, one for value
  • Cost comparison: Set two cost expressions equal to find break-even
  • Rate problems: distance = rate √ó time for each part
Desmos Power Move: Graph both equations and click the intersection point. This is often the fastest method for systems on the PSAT ‚Äî especially when the algebra looks messy.

The One Rule That Changes Everything

When you multiply or divide by a NEGATIVE number, FLIP the inequality sign. This is the #1 most-tested inequality rule on the PSAT. Everything else works exactly like equations.

Graphing

  • Number line: Open circle for < or > (not included), closed circle for ‚â§ or ‚â• (included)
  • Coordinate plane: Dashed line for < or >, solid line for ‚â§ or ‚â•. Shade the solution side.
  • Systems of inequalities: The solution is the overlapping shaded region of ALL inequalities.

Compound Inequalities

  • "and" = intersection (both true) ‚Äî narrower solution
  • "or" = union (either true) ‚Äî wider solution
Desmos Power Move: Type any inequality directly (like y > 2x + 1) and Desmos automatically shades the solution region. For systems of inequalities, enter multiple ‚Äî the overlap is your answer.

Summary

Key Takeaways

  1. Algebra is ~35% of the PSAT ‚Äî the single largest domain. Time invested here pays off the most.
  2. Know all three forms of linear equations: slope-intercept (y = mx + b), point-slope (y ‚àí y‚ÇÅ = m(x ‚àí x‚ÇÅ)), standard (Ax + By = C).
  3. Parallel lines = same slope. Perpendicular lines = negative reciprocal slopes.
  4. Systems: substitution when a variable is isolated, elimination when coefficients align, Desmos for speed.
  5. When variables cancel in a system: true statement = infinite solutions, false statement = no solution.
  6. Inequalities: FLIP the sign when multiplying/dividing by a negative.
  7. Always re-read what the question is actually asking for before choosing your answer ‚Äî the #1 algebra trap on the PSAT.
