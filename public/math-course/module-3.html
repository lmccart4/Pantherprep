Quadratics, Polynomials & Exponents

The backbone of Advanced Math — master these and you control ~35% of the test.

⏱ 2-3 sessions📋 4 topics🔨 Phase 2: Build
Quadratics are the most tested topic in Advanced Math. If you learn nothing else in this module, learn the three quadratic forms and when to use each one. That single skill can be worth 40-60 points.
Topic 3A

Quadratic Equations — The Three Forms

The SAT loves quadratics because they can be expressed in three different forms — and each form reveals different information. The key skill is knowing which form to use for each question.

Standard Form
f(x) = ax² + bx + c
Reveals: y-intercept (c)
Direction (a > 0 opens up)
Factored Form
f(x) = a(x − r)(x − s)
Reveals: x-intercepts
(zeros/roots: r and s)
Vertex Form
f(x) = a(x − h)² + k
Reveals: vertex (h, k)
(min or max point)
Converting Between Forms
▸

Standard → Factored: Factor the expression. For x² + 5x + 6, find two numbers that multiply to 6 and add to 5 → (x + 2)(x + 3).

Standard → Vertex: Complete the square, OR use the shortcut: vertex x-coordinate = −b/2a, then plug back in for y.

Factored → Standard: FOIL (multiply it out).

Any form → Solutions: Use the Quadratic Formula if factoring is difficult.

x = (−b ± √(b² − 4ac)) / 2a
The Quadratic Formula — works for ALL quadratics in standard form
The Discriminant: How Many Solutions?
▸

The expression under the square root — b² − 4ac — is called the discriminant. It tells you how many real solutions:

  • b² − 4ac > 0 → Two distinct real solutions (parabola crosses x-axis twice)
  • b² − 4ac = 0 → Exactly one real solution (parabola touches x-axis at vertex)
  • b² − 4ac < 0 → No real solutions (parabola doesn't cross x-axis)

The SAT frequently asks: "How many solutions does the equation have?" or "For what value of k does the equation have no real solutions?" Both are discriminant questions.

Vertex Shortcut (Faster Than Completing the Square)
▸

For any quadratic in standard form f(x) = ax² + bx + c:

Vertex x: x = −b / 2a
Vertex y: Plug that x back into f(x)
Min or Max? a > 0 → vertex is minimum. a < 0 → vertex is maximum.

This is MUCH faster than completing the square. When the SAT asks for the minimum or maximum value of a quadratic — that's the vertex y-coordinate.

Preview: Transformations of Quadratics
▸

Understanding vertex form is actually your first lesson in function transformations (covered fully in Module 4). The vertex form f(x) = a(x − h)² + k encodes three transformations of the parent function y = x²:

  • h shifts the parabola horizontally (right if h > 0, left if h < 0)
  • k shifts the parabola vertically (up if k > 0, down if k < 0)
  • a stretches/compresses AND reflects: |a| > 1 = narrower, |a| < 1 = wider, a < 0 = flipped

This same pattern applies to ALL functions on the SAT — not just quadratics. If you get this intuition now, Module 4 will be much easier.

Desmos Power Move: Graph any quadratic and Desmos labels the vertex, x-intercepts, and y-intercept. Click them for exact coordinates. Use sliders to explore how changing a, b, or c transforms the parabola.

Topic 3B

Polynomials & Factoring

Five Factoring Techniques
▸
  • 1. GCF (Greatest Common Factor): Always check first. Factor out the largest shared term.
    Example: 6x³ + 9x² = 3x²(2x + 3)
  • 2. Difference of Squares: a² − b² = (a + b)(a − b)
    Example: x² − 49 = (x + 7)(x − 7)
  • 3. Perfect Square Trinomial: a² + 2ab + b² = (a + b)²
    Example: x² + 6x + 9 = (x + 3)²
  • 4. Standard Trinomial: Find two numbers that multiply to ac and add to b.
    Example: x² + 5x + 6 → numbers 2 and 3 → (x + 2)(x + 3)
  • 5. Factor by Grouping: For 4-term polynomials, group pairs and factor each pair.
    Example: x³ + 3x² + 2x + 6 = x²(x + 3) + 2(x + 3) = (x² + 2)(x + 3)
Zeros, Factors & End Behavior
▸

Zeros = x-intercepts = roots = solutions. These are all the same thing: values where f(x) = 0.

If (x − r) is a factor, then r is a zero. And vice versa.

Remainder Theorem

When p(x) is divided by (x − a), the remainder = p(a). If p(3) = 0, then (x − 3) is a factor.

End Behavior

  • Even degree, positive leading coefficient: both ends UP ↑↑
  • Even degree, negative: both ends DOWN ↓↓
  • Odd degree, positive: left DOWN, right UP ↓↑
  • Odd degree, negative: left UP, right DOWN ↑↓

Exponential Growth vs. Decay
▸
Growth: f(x) = a(1 + r)ˣ — multiplier > 1
Decay: f(x) = a(1 − r)ˣ — multiplier between 0 and 1

Key difference from linear: Linear = constant amount of change (add 5 each time). Exponential = constant percent change (grow by 10% each time).

SAT tip: If a table shows values increasing by the same amount → linear. If values increase by the same multiplier → exponential.

Watch for the base: If the base is between 0 and 1, it's decay. If the base is greater than 1, it's growth. Example: 500(0.85)ᵗ → the base 0.85 is less than 1, so this is 15% decay per period.

Topic 3D

Rational Expressions & Equations

Simplifying Rational Expressions
▸

A rational expression is a fraction with polynomials. To simplify:

  • 1. Factor both numerator and denominator completely.
  • 2. Cancel any common factors.

Example: (x² − 9) / (x² + 5x + 6) = (x+3)(x−3) / (x+2)(x+3) = (x−3) / (x+2)

Domain restrictions: Any value making the denominator zero is excluded. Above: x ≠ −2 and x ≠ −3.

Solving Rational Equations
▸
  • 1. Find the LCD of all fractions.
  • 2. Multiply every term by the LCD to clear fractions.
  • 3. Solve the resulting equation.
  • 4. CHECK for extraneous solutions — plug back in to make sure no denominator = 0.

Example: 1/x + 1/3 = 1/2 → LCD = 6x → 6 + 2x = 3x → x = 6 ✓

SAT Trap: Rational equation questions almost always include an extraneous solution — an answer that makes a denominator zero. Always check your work.

Math Trap Taxonomy

Four Traps That Recur in Advanced Math

Learn these by name. When you spot one in a problem, you'll know exactly what to watch for.

1
🎯 The Wrong Target
Solved for x when they asked for x² + 1. Always underline exactly what the question asks for.
Asked: "What is 2x − 3?" You solved: x = 5, picked 5. Answer: 7
2
👻 The Distribution Ghost
Forgot to distribute to ALL terms when expanding. Especially dangerous with negative signs and squared binomials.
(x + 3)² = x² + 6x + 9 (not x² + 9)
3
🔀 The One-Case Trap
Only found one solution when there were two. Quadratics can have two roots — check both.
x² = 25 → x = 5 AND x = −5 (two answers)
4
📐 Form Mismatch
Used the wrong quadratic form for the question. Vertex form for intercepts? Standard form for the vertex? Wrong tool, slow solve.
Asked for zeros → use factored form, not vertex form

Key Takeaways
✓ Know the three quadratic forms and what each reveals: standard (y-intercept), factored (zeros), vertex (min/max).
✓ Choose the right form for each question — this is the #1 Advanced Math skill on the SAT.
✓ The discriminant (b² − 4ac) tells you how many solutions — positive = 2, zero = 1, negative = 0.
✓ Vertex x-coordinate shortcut: x = −b/2a. The y-coordinate is always the min or max value.
✓ Master all five factoring techniques — GCF, difference of squares, perfect square, trinomial, grouping.
✓ Memorize the exponent rules cold. They show up in almost every Advanced Math question.
✓ Growth vs. decay: base > 1 = growth, 0 < base < 1 = decay. Watch for (1 + r) vs. (1 − r).
✓ On rational equations, ALWAYS check for extraneous solutions.
