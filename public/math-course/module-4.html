Module 4

Functions & Their Properties

Functions are the language of Advanced Math. Learn to read, transform, compose, and solve them.

⏱ 2–3 sessions📋 4 topics🔨 Phase 2: Build
Functions are everywhere in Advanced Math. Almost every Advanced Math question involves functions in some form. The SAT tests whether you can evaluate them, transform them, compose them, and find where they intersect. This module builds the skills that connect everything in Modules 2–3 into a unified toolkit.
Big Picture

Why Functions Matter

Input (Domain)
x
→ f →
Output (Range)
f(x)

A function is a machine: you put in an input (x), it runs a rule, and you get an output (f(x)). Domain = all allowed inputs. Range = all possible outputs.

Topic 4A

Function Notation & Evaluation

4A
Reading f(x) Notation
Substitution, evaluation from equations, tables, and graphs
▸

f(x) = 2x + 3 is NOT f times x. It means "the function f, applied to input x."

f(5) means: substitute 5 for x → 2(5) + 3 = 13

f(a + 1) means: substitute (a + 1) for every x → 2(a + 1) + 3 = 2a + 5

f(x) = 0 means: find which x values make the output zero (the zeros/roots)

You can evaluate functions from equations, tables, or graphs. On a graph, f(3) means "go to x = 3 and read the y-value."

4A
Domain & Range
Restrictions, contextual domains, identifying range
▸

Domain: All x-values for which the function is defined.

Range: All y-values the function actually outputs.

Common domain restrictions on the SAT:

  • Fractions: Denominator ≠ 0
  • Square roots: Expression under radical ≥ 0
  • Context: If x represents time or quantity, x ≥ 0

For range, think about what y-values are possible. A parabola opening up with vertex at (2, −3) has range y ≥ −3.

4A
Interpreting Function Values in Context
Initial values, rates, and what constants represent
▸

The SAT loves word problems where functions model real situations. Key translations:

  • f(0) = the initial/starting value
  • The slope (or rate of change) = the "per" value (per hour, per item, etc.)
  • f(x) = 0 = when does the quantity reach zero?
  • "What does the 12 represent in f(t) = 12 + 3t?" → The initial value of 12 (the y-intercept)
  • "What does the 3 represent?" → The rate of change: it increases by 3 per unit of t

Topic 4B

Function Transformations

The SAT tests whether you can predict how a graph changes when the equation changes. Memorize this table — it covers every transformation the SAT tests.

TransformationEquation ChangeGraph Effect
Shift UP kf(x) + k↑ Moves graph UP k units
Shift DOWN kf(x) − k↓ Moves graph DOWN k units
Shift RIGHT hf(x − h)→ Moves graph RIGHT h units
Shift LEFT hf(x + h)← Moves graph LEFT h units
Vertical stretcha·f(x), a > 1↕ Steeper / taller
Vertical compressa·f(x), 0 < a < 1↔ Flatter / shorter
Reflect over x-axis−f(x)⇅ Flips upside down
Reflect over y-axisf(−x)⇄ Flips left-right
The #1 Trap: Horizontal transformations work backward. f(x − 3) shifts RIGHT 3, not left. Think of it as: the input needs to be 3 bigger to get the same output, so the graph moves right.
Desmos Tip: Graph f(x) = x² on line 1, then type (x − 3)² on line 2. Watch the parabola shift right. Change numbers to build intuition for every transformation type.

Topic 4C

Composition of Functions

4C
f(g(x)) — The Inside-Out Rule
Evaluating, composing algebraically, order matters
▸

f(g(x)) means: apply g first, then apply f to the result.

Think of it as two machines in sequence: x goes into g, the output goes into f.

Example: If f(x) = x² and g(x) = x + 3:

f(g(2)) → g(2) = 2 + 3 = 5 → f(5) = 5² = 25

g(f(2)) → f(2) = 2² = 4 → g(4) = 4 + 3 = 7

Notice: f(g(x)) ≠ g(f(x)) in general! Order matters.

To find f(g(x)) algebraically: Replace every x in f's formula with the entire expression for g(x).

f(g(x)) = f(x + 3) = (x + 3)² = x² + 6x + 9

SAT Pattern: Composition questions usually give you two simple functions and ask for f(g(specific number)). Work inside-out: evaluate g first, then plug the result into f. Don't overthink it.

Topic 4D

Nonlinear Systems

When the SAT gives you a system where one equation is linear and one is quadratic (or both nonlinear), you can solve by substitution — or graph both in Desmos.

4D
Solving Nonlinear Systems
Substitution, graphing, discriminant for intersection count
▸

Method 1: Substitution

  • Set the two equations equal: if y = x + 1 and y = x², then x + 1 = x²
  • Rearrange to standard form: x² − x − 1 = 0
  • Solve the resulting equation
  • Plug solutions back in to find y-values

Method 2: Desmos (faster!)

Graph both equations. Click the intersection points for exact coordinates.

Key insight: The discriminant of the resulting equation tells you how many intersection points exist.

4D
Systems with a Parameter
Finding k for one solution, tangent lines
▸

A harder SAT question type: "For what value of k does the system have exactly one solution?"

Strategy: Set the equations equal, rearrange to standard form, then set the discriminant = 0.

Desmos Power Move: Use a slider for k and watch the line move until it's tangent to the parabola.

Desmos Bridge: For every nonlinear system problem, try solving it BOTH algebraically and by graphing in Desmos. In Module 7, you'll learn when Desmos is faster — but building the habit of verifying starts here.
Math Trap Taxonomy

Four Traps That Recur in Function Questions

Learn these by name. When you spot one in a problem, you'll know exactly what to watch for.

1
🔄 Horizontal Flip
Confused the direction of a horizontal transformation. f(x − 3) shifts RIGHT, not left. The sign is opposite to what most students expect.
f(x − 3) → shifts RIGHT 3, f(x + 3) → shifts LEFT 3
2
🎯 Wrong Target
Solved for x when they asked for f(x), or found f(x) when they asked for x. Always underline exactly what the question asks for.
Asked: "What is f(3)?" You found x = 3, picked 3. Answer: f(3) = 7
3
🎭 Imposter Domain
Forgot a domain restriction. Especially dangerous with fractions (denominator ≠ 0) and square roots (radicand ≥ 0).
f(x) = 1/(x−3): domain is all reals except x = 3, not all reals
4
📐 Composition Swap
Evaluated f(g(x)) as g(f(x)) — or the reverse. Order matters in composition. Always work inside-out.
f(g(2)) ≠ g(f(2)) in general. Check which function is inside.

Key Takeaways
✓ f(x) means "apply function f to input x" — substitute x everywhere in the formula.
✓ Domain = allowed inputs, Range = possible outputs. Watch for division by zero and negative square roots.
✓ Horizontal transformations work backward: f(x − 3) shifts RIGHT, f(x + 3) shifts LEFT.
✓ Composition f(g(x)): work inside-out. Evaluate g first, then plug into f.
✓ Increasing functions go up as x increases; decreasing functions go down. The SAT tests this conceptually.
✓ For nonlinear systems: set equations equal, rearrange, and use the discriminant to count solutions.
✓ In context questions, identify what the constants and coefficients REPRESENT.
